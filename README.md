# ARIA - AI Companion

An interactive AI companion featuring a 3D avatar, local TTS, and beautiful UI/UX.

## Prerequisites

- Node.js (v18+)
- Python 3.10+
- Gemini API key (for chat and vision)

## Setup

### Frontend
\\\ash
npm install
npm run dev
\\\`n
### Backend
\\\ash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
\\\`n
